A simple and user-friendly Command-Line Interface (CLI) Calculator built using Python.
It supports basic arithmetic operations and uses functions, loops, and user input handling—perfect.
Runs Continously until the user chooses to exit.
------------------------------------------------------------------------------------------------------
The program shows a menu
When you run the file, it displays options like Add Task, View Tasks, Complete Task, Delete Task, and Exit.
You choose an option using a number
The program waits for your input.
Example:
Press 1 → Add a new task
Press 2 → View all tasks
Your tasks are stored in a list while the program runs
Every time you add a task, it is saved inside a Python list called tasks.
You can update tasks by selecting their number
When viewing tasks, each task has a number.
You use that number to mark a task complete or delete it.
The program keeps running until you choose Exit
The menu repeats in a loop.
When you press 5, the program prints “Goodbye!” and stops.
